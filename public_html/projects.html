strike a simple CMS

GitHub

I started working on this project initially because I was annoyed with all the popular open source CMS options. Drupal is ridiculously slow, Joomla is clunky, and Wordpress.com's pay to play model forces you to cough up money to get any use out of the platform. Strike aims to be everything that Drupal and Wordpress aren't - a fast, clean CMS for those who just want to make websites

empty stream a clean Android music player

Why do all music players on Android suck? They either have too many ads, don't work, or were last updated in 2012 (I'm looking at you, Poweramp! empty stream aims to be a a clean, simple, Material based music player for 2014.

Team766-Attendance System timecards online

GitHub

This is a simple online timecard system which logs when a registered user checks in or checks out of a given activity. I initially made the program to track users who swiped in via a barcode scanner at Robotics meetings. Swipes are stored in a MySQL database, and an admin page allows you to view all check ins by a user.

YouTube Clone it plays videos!

It's mostly a proof of concept, but a fun one at that. Registered users can upload videos via an online form, and access them later via a uniquely generated URL. All videos are loaded and played with HTML5's new video streaming function. The site acts exactly like simple version of YouTube.

PHP Login Script an authentication framework

GitHub

Ever wanted to make prevent strangers from accessing your site? Well this is the solution for you. The login script forces anonymous users to register before accessing a protected webpage. Logins are stored via either sessions or cookies, depending on how long a user wants to stay logged in for. To aid in usability, the script uses JQuery to load error messages.

Team 766 Website just another Robotics team website

Site

I made this site for my Robotics team during Junior and Senior years of high school. It's a Drupal 7 site, with video and image galleries made with Views. The site also chronicles the history of the team, with pages about old Robots we made, and a blog (also made with Views) which archives important emails sent out during the season.
